Yet Another Short Link (YASL) App

Yes, you guessed it right! This is yet-another-short-link provider.

The difference, however, is that this is a REST API only service without a web interface.

Shorten URL

Create a shortened URL, retention time: 8hrs max, 1hr min

const requestBody = {
    long_url: data.LongUrl,
    expire_time_hours: data.ExpireTime
}

const {
    statusCode,
    headers,
    body
} = await request(postUrl, {
    method: 'POST',
    headers: {
        'content-type': 'application/json'
    },
    body: JSON.stringify(requestBody)
})

Registration

Register a new user, requires an API key granted by the admin.

const requestBody = {
    user_id: data.RandomUserId,
    email: data.RandomEmail
}

const {
    statusCode,
    body
} = await request(registerUrl, {
    method: 'POST',
    headers: {
        'x-api-key': data.AdminToken,
        'content-type': 'application/json'
    },
    body: JSON.stringify(requestBody)
})

Verification

Verify the OTP provided to activate the new user.
