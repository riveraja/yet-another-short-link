Yet Another Short Link (YASL) App

Yes, you guessed it right! This is yet-another-short-link provider.

The difference, however, is that this is a REST API only service without a web interface.

Registration

Register a new user, requires an API key granted by the admin.

$ curl -X POST -H "x-api-key:[REQUEST-API-KEY]" -d '{ "user_id": "Random User", "email": "randomuser@example.com" }' http://localhost:5173/api/register
{
    "success":true,
    "activation_code":"9l63t",
    "token":"eyJhbGciOiJIUzI1NiJ9.REDACTED.mdpxgJbcuCzE3v355xm6Wujh-dqbZaKMvN2xat4UFCo"
}

Verification

Verify the OTP provided to activate the new user.

$ curl -X GET -H "x-api-key:eyJhbGciOiJIUzI1NiJ9.REDACTED.mdpxgJbcuCzE3v355xm6Wujh-dqbZaKMvN2xat4UFCo" http://localhost:5173/api/verify=9l63t
{
    "success":true
}

Shorten URL

Create a shortened URL, retention time: 8hrs max, 1hr min

$ curl -X POST -H "x-api-key:eyJhbGciOiJIUzI1NiJ9.REDACTED.mdpxgJbcuCzE3v355xm6Wujh-dqbZaKMvN2xat4UFCo" \
    -d '{ "long_url": "https://medium.com/@tdodds/the-wild-story-behind-the-longest-website-url-in-the-world-8a6fd24bc15e", "expire_time_hours": 1 }' http://localhost:5173/api/shorten-url
{
    "success":true,
    "status":200,
    "long_url":"https://medium.com/@tdodds/the-wild-story-behind-the-longest-website-url-in-the-world-8a6fd24bc15e",
    "short_url":"http://localhost:5173/fn15f",
    "created_at":"2024-04-19T12:18:15.984Z",
    "expires_on":"2024-04-19T13:18:15.984Z"
}
